# Technology Stack

## Core
- **Framework:** Next.js 16.1.6 (App Router)
- **Language:** TypeScript 5
- **Runtime:** Node.js (Vercel compatible)

## Frontend
- **Library:** React 19.2.3
- **Styling:** Tailwind CSS 4.2.1
- **Icons:** Lucide React
- **Components:** Custom Neo-Brutalism design system

## Backend & Data
- **Database:** MongoDB (via Mongoose 9.3.1)
- **ODM:** Mongoose
- **Auth:** 
  - Firebase Authentication (Client-side / Google)
  - JWT (JSON Web Tokens) for Vendor sessions
  - BcryptJS for password hashing

## Infrastructure
- **Hosting:** Vercel (Production)
- **Image Hosting:** Cloudinary
- **Payments:** Razorpay

## Development Tools
- **Linting:** ESLint 9
- **PostCSS:** Autoprefixer 10, PostCSS 8
- **Package Manager:** npm
